Sometimes you don't want dual panels. Sometimes you don't want voice output. Sometimes you don't want thumbnails and galleries and settings panels.

Sometimes you just want to TALK.

And you know what? There's beauty in that simplicity.

Distraction-free text-based conversation:

- Large textarea for input
- Send button (or press Enter)
- Clear button (fresh start)
- Model selectors (text and image)

No sidebar. No panels. No clutter. Just CONVERSATION.

### Multiple AI Models

Even minimalism needs OPTIONS:

**Text Models:**
- **Unity AI (Custom)**: Me, with personality, uncensored

Pick the personality that matches your vibe.

### Image Generation

- "Create a dragon"

Images display right in the chat. Simple. Effective.

Minimalist doesn't mean UGLY.

Works on desktop. Works on mobile. Works on tablets. Works on whatever the fuck you're using.

## How to Use This Thing

### Basic Chat

*types slowly*

Literally couldn't be simpler:

1. Select AI model (Unity AI or Evil Mode)
2. Type message in textarea
3. Press Enter to send (Shift+Enter for new line)
4. Read response
5. Repeat

No configuration. No setup. Just TALK.

### Image Generation

Ask naturally:
- "Show me X"
- "Create Y"
- "Generate Z"

I'll generate images inline with my responses. No special syntax. No buttons. Just natural language.

### Controls

*points at interface*

Four controls. That's it:

- **Send Message Button**: Submit your message
- **Clear Chat Button**: Wipe conversation, start fresh
- **Text Model Dropdown**: Switch between Unity AI and Evil Mode
- **Image Model Dropdown**: Select image generation model

Can't get more minimal than that.

- **Font Awesome**: Icon library (icons)

Just the essentials. No bloat.

## Technical Details (Brief)

*adjusts glasses*

- Pollinations API for text and image generation
- Unity AI persona: Uncensored, creative, personality-driven
- Evil Mode: No restrictions, pure unfiltered chaos
- Direct fetch to gen.pollinations.ai/image/ (API key auth)
- Conversation messages styled with left border accent
- Auto-scrolling chat output area
- LocalStorage for model selection persistence
- Uncensored generation (safe=false parameter)
