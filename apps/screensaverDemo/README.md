Smooth cross-fade transitions between images. No jarring cuts. No loading flicker. Just BUTTER-SMOOTH transitions that look professional as fuck.

- Horizontal scrollable gallery

## Technical Details (Nerd Zone)

*adjusts glasses while smoking*

- **Auto-Prompt**: Uses Unity persona for creative, dark prompts every 20s
- **Dual-Image Buffering**: Smooth transitions without flashing
- **Image Preloading**: Next image loads before display (no loading screen)
- **LocalStorage**: History and settings persist across sessions
- **Auto-Save**: Settings saved on every change
- **Uncensored Generation**: safe=false parameter (no content restrictions)
- **Random Seeds**: Every generation is unique
