Overview & Features

Moana Miner is a complete mining solution for RavenCoin (RVN) using the KawPow algorithm. It provides an easy-to-use web dashboard for configuration and monitoring, eliminating the need to manually edit config files or run command-line miners.

Click image to expand

Moana Dashboard - Real-time mining stats with Unity AI commentary

  • Web-Based Dashboard
    Beautiful web interface at localhost:4068 for all configuration and monitoring. No command line required.
  • Real-Time Statistics
    Live hashrate, shares accepted/rejected, GPU temperature, power consumption, and more.
  • Pool & Solo Mining
    Switch between pool mining and solo mining with a single click. Pre-configured with popular pools.
  • Multiple Miner Support
    Choose between T-Rex Miner (stable, widely compatible) or Rigel Miner (optimized for RTX cards).
  • All-Time Earnings Tracking
    Track your total shares, uptime, and estimated RVN earnings across all sessions.
  • Live RVN Price
    Real-time RavenCoin price from CoinGecko displayed in the dashboard.
  • GPU Monitoring
    Core clock, memory clock, temperature, fan speed, and power draw via nvidia-smi.

Installation Guide

1

Download & Extract Moana

Download the Moana.zip file using the button above and extract it to a folder of your choice. We recommend a simple path like C:\Mining\Moana to avoid issues with spaces in paths.

Moana/
├── bin/
│ ├── Moana.bat # Main launcher
│ ├── dashboard.py # Dashboard server
│ ├── config.bat # Configuration reference
│ ├── moana_config.json # Settings (auto-created)
│ └── moana_stats.json # Stats (auto-created)
└── Moana.lnk # Desktop shortcut
2

Install Python (if not already installed)

Moana's dashboard requires Python. Download and install Python from the official website:

  • Go to python.org/downloads
  • Download Python 3.11 or later (Windows installer)
  • Important: Check "Add Python to PATH" during installation
  • Complete the installation

Verify Python is installed by opening Command Prompt and typing:

python --version

You should see something like Python 3.11.x

3

Download a Miner (T-Rex or Rigel)

Moana is a dashboard/launcher - you need to download the actual mining software separately. Choose one of the following miners:

T-Rex Miner (Recommended)

Stable, widely tested, works on most NVIDIA GPUs. 1% dev fee.
GitHub: github.com/trexminer/T-Rex
Download: github.com/trexminer/T-Rex/releases
Get the Windows CUDA version (e.g., t-rex-0.26.8-win.zip)

Rigel Miner (RTX Optimized)

Optimized for RTX 30/40 series cards, may get 1-3% higher hashrates. 1% dev fee.
GitHub: github.com/rigelminer/rigel
Download: github.com/rigelminer/rigel/releases
Get the Windows version (e.g., rigel-1.x.x-win.zip)

After downloading:

  • Extract the miner zip file
  • Copy the miner executable (t-rex.exe or rigel.exe) to the Moana/bin/miner/ folder
  • Make sure the file is named exactly t-rex.exe or rigel.exe

Antivirus Warning

Mining software is often flagged as malware by antivirus programs (false positive). You may need to add an exception for the miner executable and the Moana folder.

4

Create a RavenCoin Wallet

You need a RavenCoin wallet address to receive mining rewards. Options include:

  • Raven Core Wallet - Official desktop wallet from ravencoin.org
  • Exchange Wallet - Use your RVN deposit address from exchanges like TradeOgre, Bittrex, etc.
  • Mobile Wallet - Zelcore or Trust Wallet support RVN

Your wallet address will look something like: RUcFJm7n2W5oQnb4pLAHmL9fYEBDFSaKpk

5

Launch Moana

Double-click Moana.bat in the bin folder (or use the shortcut). This will:

  • Start the dashboard server on port 4068
  • Automatically open your browser to http://127.0.0.1:4068

You'll see the Moana dashboard where you can configure everything through the web interface!

Configuration

All configuration is done through the web dashboard. Here's what each setting does:

Wallet Address

Enter your RavenCoin wallet address. This is where your mining rewards will be sent. Double-check this address - sending to the wrong address means lost coins!

Worker Name

A name to identify this mining rig on the pool. Useful if you have multiple miners. Default is "Moana". Use only letters and numbers (no spaces).

Miner Selection

Choose which mining software to use:

  • T-Rex - Stable, works on all NVIDIA GPUs, good dev fee (1%)
  • Rigel - Newer, optimized for RTX cards, may get 1-3% higher hashrate

Make sure you've placed the correct .exe file in the bin folder!

Pool vs Solo Mining

Pool Mining: Join thousands of miners, get small frequent payouts. Recommended for most users. Your hashrate contributes to finding blocks together.

Solo Mining: Mine alone, get full block reward (2500 RVN) if YOU find a block. Very rare for small miners - only recommended if you have significant hashpower (100+ MH/s).

Checking Your Mining Stats

To see your pool statistics, payouts, and hashrate history, visit your pool's website and enter your wallet address. For example, for 2Miners go to rvn.2miners.com/account/YOUR_WALLET_ADDRESS

Using the Dashboard

Starting Mining

Once you've entered your wallet address and selected your miner, click the "Start Mining" button. The miner will launch and you'll see real-time stats appear in the dashboard.

Dashboard Stats

  • Hashrate - Your mining speed in MH/s (megahashes per second)
  • Shares Accepted - Valid work submitted to the pool
  • Shares Rejected - Invalid work (should be very low)
  • GPU Temp - Current GPU temperature (keep below 80°C)
  • Power - GPU power consumption in watts
  • Uptime - How long the miner has been running
  • RVN Price - Current RavenCoin price from CoinGecko

Stopping Mining

Click "Stop Mining" to gracefully stop the miner. Your all-time stats are automatically saved. You can close the browser tab - the miner will keep running. To fully stop, use the Stop button or close the command prompt window.

Troubleshooting

"Python not found"

Python is not installed or not in your PATH. Reinstall Python and make sure to check "Add Python to PATH" during installation. Restart your computer after installing.

"Miner not found"

The miner executable (t-rex.exe or rigel.exe) is not in the bin folder. Download the miner from the links above and copy the .exe file to Moana/bin/.

Dashboard not opening

Manually open your browser and go to http://127.0.0.1:4068. If it still doesn't work, check if another program is using port 4068, or if your firewall is blocking it.

Low hashrate or GPU not detected

Make sure you have the latest NVIDIA drivers installed. Try running as Administrator. Check that your GPU supports CUDA and has enough VRAM (4GB minimum for KawPow).

Antivirus blocks/deletes miner

Mining software triggers false positives. Add an exception for the entire Moana folder in your antivirus settings. For Windows Defender: Settings → Virus & threat protection → Manage settings → Exclusions → Add folder.

Frequently Asked Questions

How much can I earn mining RavenCoin?
Earnings depend on your hashrate, electricity costs, and RVN price. A typical RTX 3060 gets around 15-20 MH/s. Use online calculators like WhatToMine.com to estimate profitability. Mining is often not profitable with high electricity costs.
Is this safe? Will it damage my GPU?
Mining puts your GPU under sustained load, which generates heat. Keep temperatures below 80°C (ideally below 70°C) and ensure good airflow. Mining doesn't inherently damage GPUs, but prolonged high temperatures can reduce lifespan. Consider undervolting for efficiency.
Why is my hashrate lower than expected?
KawPow is memory-intensive. Ensure you have fast memory clocks and consider overclocking. Also check: latest drivers, no background programs using GPU, power limit not too low, thermal throttling (GPU too hot).
When do I get paid from the pool?
Each pool has a minimum payout threshold. For 2Miners it's 10 RVN. Once your balance reaches the threshold, the pool automatically sends to your wallet. Check the pool website for your balance and payout history.
Can I mine on multiple GPUs?
Yes! Both T-Rex and Rigel automatically detect and use all available NVIDIA GPUs. The dashboard will show combined hashrate. Each GPU's stats may be shown separately depending on the miner.
Should I use pool or solo mining?
Pool mining for most users. Solo mining only makes sense with very high hashrate (100+ MH/s) and patience. With solo mining you could go months without finding a block, but when you do, you get the full 2500 RVN reward.
